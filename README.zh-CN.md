# CHC5223-Terminology-List

[English](./README.md) · [繁體中文](./README.zh-HK.md) · **简体中文**</div>

## 簡介
这个项目旨在为CHC5223(数据结构与算法)课程创建术语表。该术语表将收集Leon Liang教授在课堂上提到的所有术语和重要词汇。

## 如何使用
前往[Releases](./release)并下载最新的 .xlsx 文档，每周的术语都会在一个单独的表格中。
